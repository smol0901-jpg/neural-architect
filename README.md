# 🧠 NEURAL ARCHITECT PREMIUM++

## AI-First Site Engine v0.1

**Гидро-движок** нового поколения для создания сайтов с нейро-вайб эффектами, глитчем и кибер-атмосферой.

---

## 🚀 Возможности

- **Нейро-вайб эффекты** — плавные анимации с эффектом "глитча" каждые 4 секунды
- **Кибер-атмосфера** — тёмная тема с градиентами cyan → purple
- **Премиум-рендеринг** — высококачественный визуальный стиль
- **Адаптивность** — работает на всех устройствах
- **API-связь** — все компоненты связаны через единое API

---

## 📦 Структура проекта

```
neural-architect/
├── app/
│   ├── page.tsx          # Главная страница
│   ├── layout.tsx       # Layout с мета-тегами
│   └── globals.css      # Глобальные стили
├── components/
│   ├── NeuralTitle.tsx  # Компонент названия с нейро-вайб
│   ├── NeuralParticles.tsx  # Фоновые нейро-частицы
│   └── NeuralButton.tsx # Кнопки с эффектами
├── public/
│   └── index.html       # HTML-индекс для визуального рендеринга
└── package.json
```

---

## 🛠 Установка

```bash
npx create-next-app@latest neural-architect --typescript --tailwind --eslint --app --yes
cd neural-architect
npm run dev
```

---

## 🎨 Компоненты

### NeuralTitle
Основной компонент названия с эффектом нейро-вайба:
- Глитч-эффект каждые 4 секунды
- Градиенты cyan → purple
- Адаптивный размер

### NeuralParticles
Фоновые нейро-частицы для атмосферы:
- Анимированные частицы
- Плавное движение
- Кибер-эстетика

### NeuralButton
Интерактивные кнопки с эффектами:
- Градиентные кнопки
- Hover-анимации
- Активные состояния

## 📱 HTML Индекс
